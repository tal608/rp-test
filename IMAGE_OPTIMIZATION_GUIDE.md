## Implementation Notes

### SEO Filename Format
All filenames follow this pattern:
- Descriptive keywords (happy-golden-retriever)
- Service type (dog-hike, dog-grooming, pet-transport)
- Location (waunakee-wi, madison-wi, etc.)
- Brand name (river-paws)
- File extension (.jpg)

### Alt Text Guidelines
- Written in complete sentence form
- Includes location (city, state)
- Mentions River Paws brand name
- Describes the action/emotion
- Natural and conversational tone
