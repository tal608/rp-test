# Caretaker Images

This directory should contain the caretaker images from the live River Paws website.

## Required Images

Please download the following images from https://www.riverpaws.dog/caretakers/ and save them with these exact filenames:

- `amanda-masarik.jpg` - Amanda Masarik (Owner)
- `kelly-esterholm.jpg` - Kelly Esterholm (Dog Groomer)
- `stephanie-helt.jpg` - Stephanie Helt (Dog Groomer)
- `chloe-lane.jpg` - Chloe Lane (Adventure Out Dog Hikes)
- `elaina-katzke.jpg` - Elaina Katzke (Dog Groomer)
- `france-lozano.jpg` - France Lozano (Dog Groomer)

## How to Download

1. Visit https://www.riverpaws.dog/caretakers/ in your web browser
2. Right-click on each caretaker's photo
3. Select "Save image as..." or "Copy image"
4. Save the image to this directory with the corresponding filename listed above

## Image Requirements

- Format: JPG or PNG
- Recommended size: At least 800x600 pixels
- The images should be clear and professional headshots/portraits of each team member
