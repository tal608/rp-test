# Gallery Images - Optimization Guide

## Best Image Types for Customer Conversion

Based on marketing research, these image types convert best:

### 1. **Before/After Grooming Transformations** ⭐ HIGHEST CONVERSION
- Show dramatic results
- File naming: `grooming-before-after-[breed]-[location].jpg`
- Example: `grooming-before-after-golden-retriever-waunakee.jpg`
- Alt text: Include "before and after dog grooming" + location keywords

### 2. **Happy Dogs After Grooming** ⭐ HIGH CONVERSION
- Dogs looking relaxed, satisfied, and beautiful
- Shows results and happy customers
- File naming: `happy-dog-after-grooming-[breed].jpg`

### 3. **Multiple Dogs on Hikes** ⭐ HIGH CONVERSION
- Social proof - shows groups having fun
- Adventure appeal
- File naming: `hiking-adventure-group-dogs-[location].jpg`

### 4. **Dogs During Grooming (Calm & Happy)** ⭐ MEDIUM CONVERSION
- Shows trust and comfort
- Demonstrates gentle approach
- File naming: `calm-dog-during-grooming-[breed].jpg`

### 5. **Finished Grooming Close-ups** ⭐ MEDIUM CONVERSION
- Details of quality work
- Professional results
- File naming: `grooming-detail-[service]-[breed].jpg`

## SEO Optimization Checklist

For each image you add:

1. **File Name** (MOST IMPORTANT for SEO):
   - Use descriptive, keyword-rich names
   - Format: `[service]-[subject]-[location]-[descriptor].jpg`
   - Example: `dog-grooming-golden-retriever-waunakee-before-after.jpg`
   - Include location: waunakee, madison, middleton, deforest, sun-prairie

2. **Image Size**:
   - Width: 1200-2000px (optimal for web)
   - File size: Under 500KB (use compression)
   - Format: JPEG for photos (PNG only if transparency needed)

3. **Alt Text** (Auto-generated from file structure):
   - Include: service type + location + context
   - Example: "Professional dog grooming service at River Paws in Waunakee, Wisconsin showing before and after transformation"
   - Include surrounding areas: "serving Madison, Middleton, DeForest, and Sun Prairie"

4. **Title**:
   - Customer-facing, engaging titles
   - Examples: "Transformation Magic", "Spa Day Bliss", "Happy & Groomed"

## How to Add Images from Facebook

1. Visit https://www.facebook.com/riverpaws
2. Find high-quality posts with engaging images
3. Right-click image → "Save image as..."
4. Rename with SEO-friendly filename
5. Save to `public/gallery/`
6. Update `page.tsx` with new image entry

## Image Priority System

- **Priority = true**: Load first (above the fold, key conversion images)
- **Priority = false**: Lazy load (below the fold, supporting images)

Priority images should be:
- Before/after transformations
- Hero images that appear first
- Most engaging/high-quality content

## Recommended Image Count

- **Grooming**: 8-12 images (show variety, results, transformations)
- **Hiking**: 6-8 images (show adventure, groups, trails)
- **Playtime**: 4-6 images (show fun, socialization)

**Total: 18-26 images** - Optimal balance of content and page speed

## Image Compression

Before adding images:
1. Use online tools like TinyPNG.com or Squoosh.app
2. Target: 300-500KB per image
3. Maintain quality at 80-85% compression
4. Test on mobile - ensure fast loading
